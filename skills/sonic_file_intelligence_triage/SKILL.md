## Complete Showtech File Reference

### **A. Platform and System Information Files**

#### **`version` or `show version`**
- **Purpose**: SONiC OS version, build information, kernel version
- **Content**: SONiC version string, build date, kernel version, hardware platform
- **Used For**: Version compatibility checks, bug identification, feature support validation
- **Key Info**: `SONiC-OS-4.5.1-Enterprise_Standard`, kernel version, build timestamp
- **Troubleshooting**: Version-specific issues, compatibility problems

#### **`platform` or `show platform`**
- **Purpose**: Hardware platform identification and capabilities
- **Content**: Platform name, hardware SKU, ASIC type, serial number
- **Used For**: Platform-specific troubleshooting, hardware compatibility
- **Key Info**: `x86_64-dell_s5448f-r0`, product name, ASIC information
- **Troubleshooting**: Platform-specific issues, hardware compatibility problems

#### **`inventory` or `show inventory`**
- **Purpose**: Hardware component inventory and status
- **Content**: List of all hardware components with serial numbers and status
- **Used For**: Hardware tracking, warranty information, component replacement
- **Key Info**: Chassis, power supplies, fans, transceivers, ASIC modules
- **Troubleshooting**: Hardware failures, component issues

#### **`environment` or `show environment`**
- **Purpose**: Environmental monitoring (temperature, voltage, fans)
- **Content**: Temperature readings, voltage levels, fan speeds, power status
- **Used For**: Environmental monitoring, hardware health assessment
- **Key Info**: Temperature thresholds, voltage ranges, fan RPMs
- **Troubleshooting**: Overheating issues, power supply problems

### **B. Network Interface Files**

#### **Interface Configuration Files**
- **Purpose**: Interface configuration and status
- **Content**: Interface settings, IP addresses, VLAN assignments, MTU
- **Used For**: Interface configuration, network troubleshooting
- **Key Info**: Interface names, IP addresses, VLAN IDs, MTU settings
- **Troubleshooting**: Interface connectivity, configuration errors

#### **`lldp_neighbors`**
- **Purpose**: LLDP neighbor discovery information
- **Content**: Connected devices, port information, capabilities
- **Used For**: Network topology discovery, connectivity verification
- **Key Info**: Neighbor device names, port connections, device capabilities
- **Troubleshooting**: Network connectivity, LLDP issues

### **C. Routing Protocol Files**

#### **`bgp_summary`**
- **Purpose**: BGP session summary and status
- **Content**: BGP neighbor states, session uptime, route counts
- **Used For**: BGP monitoring, connectivity troubleshooting
- **Key Info**: Neighbor IPs, session states, uptime, route counts
- **Troubleshooting**: BGP session failures, connectivity issues

#### **`bgp_routes`**
- **Purpose**: BGP routing table information
- **Content**: BGP routes, next hops, AS paths, attributes
- **Used For**: Routing analysis, BGP troubleshooting
- **Key Info**: Route prefixes, next hops, AS paths, communities
- **Troubleshooting**: Routing issues, BGP problems
